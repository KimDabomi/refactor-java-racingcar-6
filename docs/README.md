# ✅ 게임 방법
- 자동차마다 이름을 갖고 있음, 전진 여부를 출력할 때 자동차 이름을 같이 출력
- 자동차 이름은 쉼표로 구분, 5자 이하만 가능
- 사용자가 몇 번의 이동을 할 것인지 입력
- 전진 조건
  - 0에서 9사이의 무작위 값을 구한 후 무작위 값이 4이상인 경우
- 공동 우승인 경우 쉼표를 이용하여 구분
